<h1>Rick and Morty Simulator </h1> 
Simple web application that displays the characters in the Rick and Mory world at random using the Rick and Morty API.

<h2> Technologies Used </h2> 
* HTML5
* CSS3
* JavaScript
* jQuery
* Ajax
* Rick and Morty API
